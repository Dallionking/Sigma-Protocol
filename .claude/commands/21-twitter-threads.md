## 🧠 2026 Twitter/X Thread Science

### What The Algorithm Rewards

**Engagement Signals (In Order):**
1. Replies (highest weight)
2. Reposts/Retweets
3. Bookmarks (growing in importance)
4. Likes
5. Click-throughs
6. Dwell time

### Thread Performance Factors

| Factor | Impact |
|--------|--------|
| **Opening hook** | Critical - Determines if people click "Show thread" |
| **First 3 tweets** | High - Early engagement triggers distribution |
| **Thread length** | 5-10 tweets optimal; up to 15 for deep dives |
| **Visual breaks** | High - Images/charts every 3-4 tweets |
| **Reply engagement** | Very high - Author replies boost visibility |

## 📋 Command Usage

```bash
@21-twitter-threads
@21-twitter-threads --length=medium --type=educational
@21-twitter-threads --type=story --cta=follow
@21-twitter-threads --batch=3
```

| Parameter | Description | Default |
|-----------|-------------|---------|
| `--length` | short (5-7), medium (8-10), long (11-15) tweets | `medium` |
| `--type` | educational, story, list, breakdown, controversial | `educational` |
| `--cta` | follow, reply, repost, link | `follow` |
| `--batch` | Number of threads to generate | `1` |

## 🧵 THREAD STRUCTURES

### Structure A: The Educational Breakdown (Most Common)

```
┌────────────────────────────────────────────────────────────┐
│ TWEET 1: Hook + Promise                                    │
│ "[Result/Claim]. Here's how:" or "Here's the breakdown:"   │
├────────────────────────────────────────────────────────────┤
│ TWEET 2: Context or "Why this matters"                     │
│ Set the stage, add credibility                             │
├────────────────────────────────────────────────────────────┤
│ TWEETS 3-8: The meat (one point per tweet)                 │
│ "1/ [Point]" numbered for clarity                          │
├────────────────────────────────────────────────────────────┤
│ TWEET 9: Summary or "Key takeaway"                         │
│ Bring it all together                                      │
├────────────────────────────────────────────────────────────┤
│ TWEET 10: CTA                                              │
│ "Follow for more" + Engagement ask                         │
└────────────────────────────────────────────────────────────┘
```

### Structure B: The Story Thread

```
┌────────────────────────────────────────────────────────────┐
│ TWEET 1: Hook at the climax                                │
│ Start at the most dramatic moment                          │
├────────────────────────────────────────────────────────────┤
│ TWEET 2: "Let me back up..."                               │
│ Take them to the beginning                                 │
├────────────────────────────────────────────────────────────┤
│ TWEETS 3-7: The journey                                    │
│ Build tension, show struggle                               │
├────────────────────────────────────────────────────────────┤
│ TWEET 8: The turning point                                 │
│ What changed everything                                    │
├────────────────────────────────────────────────────────────┤
│ TWEET 9: The lesson                                        │
│ What you learned                                           │
├────────────────────────────────────────────────────────────┤
│ TWEET 10: CTA                                              │
│ Apply the lesson + engagement                              │
└────────────────────────────────────────────────────────────┘
```

### Structure C: The Controversial Take

```
┌────────────────────────────────────────────────────────────┐
│ TWEET 1: Bold claim that sparks debate                     │
│ "Unpopular opinion:" or direct hot take                    │
├────────────────────────────────────────────────────────────┤
│ TWEET 2: "Here's why:"                                     │
│ Start building your argument                               │
├────────────────────────────────────────────────────────────┤
│ TWEETS 3-6: Evidence and reasoning                         │
│ Make your case                                             │
├────────────────────────────────────────────────────────────┤
│ TWEET 7: Acknowledge the counterargument                   │
│ "I know what you're thinking..."                           │
├────────────────────────────────────────────────────────────┤
│ TWEET 8: Why you still believe it                          │
│ Reinforce your position                                    │
├────────────────────────────────────────────────────────────┤
│ TWEET 9-10: CTA + Engagement                               │
│ "Agree or disagree?" drive replies                         │
└────────────────────────────────────────────────────────────┘
```
