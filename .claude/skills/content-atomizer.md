# Content Atomizer Skill

Transform one piece of content into many. Take a blog post, video transcript, or podcast and atomize it into platform-optimized social posts, emails, threads, and more.

> **"Create once, distribute everywhere."**

## When to Invoke

Invoke this skill when:
- Repurposing long-form content into social media
- Creating content calendars from existing assets
- Maximizing ROI on content creation
- Adapting content for different platforms
- Building content flywheel systems

## 🎥 YouTube Atomization

### YouTube Shorts Script (< 60 sec)

```markdown
[HOOK: First 1-2 sec]
[Movement + bold statement]
"Stop doing [thing] wrong!"

[CONTENT: 3-45 sec]
"Here's the right way:"
[Step 1 — quick demo or explanation]
[Step 2 — quick demo or explanation]
[Step 3 — quick demo or explanation]

[PAYOFF: 45-55 sec]
"And that's how you [achieve result]"

[CTA: 55-60 sec]
"Follow for more tips like this"
```

### Long-Form Clips

Extract clips based on:
- Highest-energy moments
- Controversial statements
- Clear explanations
- Emotional peaks
- Funny moments

Each clip should:
- Stand alone (context within)
- Have strong hook (first 3 sec)
- Be 15-90 seconds
- End with value delivered

## 🔄 Atomization Best Practices

### DO:
- ✅ Maintain brand voice across all formats
- ✅ Optimize for each platform's native style
- ✅ Lead with value, not promotion
- ✅ Create platform-native content (not just copy-paste)
- ✅ Track which formats perform best
- ✅ Batch create all atomized content at once

### DON'T:
- ❌ Post identical content everywhere
- ❌ Ignore platform-specific norms
- ❌ Spam all platforms same day
- ❌ Forget CTAs back to original
- ❌ Over-promote without value

### Platform Posting Frequency

| Platform | Optimal Frequency |
|----------|------------------|
| Twitter/X | 3-5x daily |
| LinkedIn | 1-2x daily |
| Instagram Feed | 1x daily |
| Instagram Stories | 3-5x daily |
| TikTok | 1-3x daily |
| YouTube Shorts | 1x daily |
| Email | 1-3x weekly |
