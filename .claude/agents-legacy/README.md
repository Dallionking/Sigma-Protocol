# Legacy Agents Directory

**Status:** DEPRECATED as of v5.0

This directory contains the old agent files that were referenced by command wrappers.

As of v5.0, all step commands are self-contained in:
- `templates/steps/` (canonical source)
- `.claude/commands/` (synced from templates)

These legacy files are kept for reference only. Do not use `@agent-name` syntax anymore.
