## 📋 Command Usage

```bash
@23-story-posts
@23-story-posts --platform=instagram --type=behind-scenes
@23-story-posts --platform=all --batch=5
```

### Parameters

| `--type` | behind-scenes, poll, question, announcement, value, personal | `value` |
| `--batch` | Number of story sequences | `1` |

## 📱 STORY PSYCHOLOGY

### Why Stories Matter

**Engagement Hierarchy:**
1. Stories feel personal (1-on-1 conversation)
2. Stories are time-limited (FOMO factor)
3. Stories reward consistent viewers
4. Stories build parasocial relationships

**What Works:**
- Authenticity over polish
- Questions and polls drive interaction
- Behind-the-scenes builds trust
- Value-packed tips get saved
- Personal moments create connection

## 🎬 STORY STRUCTURES

### Structure A: The Value Sequence (5-7 Frames)

```
┌────────────────────────────────────────────────────────────┐
│ FRAME 1: Hook - Bold claim or question                     │
├────────────────────────────────────────────────────────────┤
│ FRAME 2: Context - Why this matters                        │
├────────────────────────────────────────────────────────────┤
│ FRAMES 3-5: Value - Tips, steps, or insights               │
├────────────────────────────────────────────────────────────┤
│ FRAME 6: Summary - Key takeaway                            │
├────────────────────────────────────────────────────────────┤
│ FRAME 7: CTA - DM, reply, link, save                       │
└────────────────────────────────────────────────────────────┘
```

### Structure B: The Behind-the-Scenes (3-5 Frames)

```
┌────────────────────────────────────────────────────────────┐
│ FRAME 1: Setup - "Currently working on..."                 │
├────────────────────────────────────────────────────────────┤
│ FRAMES 2-3: Process - Show the messy middle                │
├────────────────────────────────────────────────────────────┤
│ FRAME 4: Insight - What you're learning/feeling            │
├────────────────────────────────────────────────────────────┤
│ FRAME 5: Question - Ask for their thoughts                 │
└────────────────────────────────────────────────────────────┘
```

### Structure C: The Engagement Sequence (3-4 Frames)

```
┌────────────────────────────────────────────────────────────┐
│ FRAME 1: Question/Poll setup                               │
├────────────────────────────────────────────────────────────┤
│ FRAME 2: The actual poll/quiz/question sticker             │
├────────────────────────────────────────────────────────────┤
│ FRAME 3: Your take/opinion                                 │
├────────────────────────────────────────────────────────────┤
│ FRAME 4: Results tease or follow-up                        │
└────────────────────────────────────────────────────────────┘
```

## 📱 INSTAGRAM STORIES

### Best Practices
- 15 seconds max per frame (or quick-tap)
- Use native features (polls, questions, sliders)
- Vertical format only (9:16)
- Music increases engagement 15-20%
- Post 3-7 stories/day for optimal reach

### Story Types That Work

**1. Question Box Prompts**
```
FRAME 1:
[VISUAL: Your face or relevant background]
[TEXT: "Let's play a game 👇"]

FRAME 2:
[VISUAL: Question sticker]
[STICKER: "What's your biggest challenge with [topic]?"]

FRAME 3:
[VISUAL: Your face responding]
[TEXT: "I'll answer the best ones in my next story!"]
```

**2. Poll Stories**
```
FRAME 1:
[VISUAL: Split comparison or your face]
[TEXT: "Quick poll:"]

FRAME 2:
[VISUAL: Poll sticker centered]
[POLL: "[Option A]" vs "[Option B]"]

FRAME 3:
[VISUAL: Your take]
[TEXT: "I said [your choice] because [reason]"]
```

**3. Value Tips Sequence**
```
FRAME 1:
[VISUAL: Attention-grabbing background]
[TEXT: "3 [topic] tips nobody talks about"]

FRAME 2:
[VISUAL: Tip 1]
[TEXT: "1️⃣ [Tip]"]

FRAME 3:
[VISUAL: Tip 2]
[TEXT: "2️⃣ [Tip]"]

FRAME 4:
[VISUAL: Tip 3]
[TEXT: "3️⃣ [Tip]"]

FRAME 5:
[VISUAL: Your face]
[TEXT: "Save this story 📌"]
```

**4. Behind-the-Scenes**

## 📺 YOUTUBE COMMUNITY POSTS

### Best Practices
- Image posts get highest engagement
- Polls drive comments and notifications
- Build anticipation for uploads
- Ask questions your audience cares about
- Link to relevant videos

### Community Post Types

**1. Poll Post**
```
📊 Quick question for you:

[Question about upcoming content or topic]

[Poll Option 1]
[Poll Option 2]
[Poll Option 3]
[Poll Option 4]

Comment WHY you voted the way you did 👇
```

**2. Behind-the-Scenes**
```
🎬 Sneak peek at what's coming...

[IMAGE: Screenshot or setup shot]

Working on a video about [topic]. 

Drop a 🔥 if you want to see this!
```

**3. Value Post (No Link)**
```
💡 Quick tip that took me years to learn:

[Valuable insight or tip]

[Expand on it in 2-3 sentences]

Save this post for later 📌
```

**4. Engagement Post**
```
🤔 I'm curious...

[Question relevant to your niche]

Be specific! I read every comment.
```

**5. Video Tease**
```
🚨 New video dropping [Day]!

[IMAGE: Thumbnail]

[One-sentence hook about the video]

Turn on 🔔 so you don't miss it!
```

## 📝 COMPLETE STORY EXAMPLES

### Example 1: Value Sequence (Instagram)

```markdown
## STORY: 3 Productivity Hacks

### FRAME 1: HOOK
**Visual:** Close-up selfie, slightly puzzled expression
**Text:** "Why does nobody talk about this? 👇"
**Music:** Upbeat, trending sound

### FRAME 2: CONTEXT
**Visual:** Quick transition to workspace
**Text:** "3 productivity hacks that changed everything for me"
**Duration:** 2 seconds

### FRAME 3: TIP 1
**Visual:** Split screen - before/after
**Text:** "1️⃣ Time block in 90-minute sprints"
**Text (small):** "Your brain can only focus for ~90 min"

### FRAME 4: TIP 2
**Visual:** Phone screen showing app
**Text:** "2️⃣ Put phone in different room"
**Text (small):** "Out of sight = out of mind"

### FRAME 5: TIP 3
**Visual:** Morning routine shot
**Text:** "3️⃣ Do hardest task first"
**Text (small):** "When willpower is highest"

### FRAME 6: SUMMARY
**Visual:** Back to face
**Text:** "Which one are you trying first?"
**Slider Sticker:** "How productive are you?" 🔥

### FRAME 7: CTA
**Visual:** Face with pointing gesture
**Text:** "DM me 'PRODUCTIVITY' for my full system"
**Sticker:** Link to resource
```

### Example 2: Engagement Sequence (Instagram)

```markdown
## STORY: This or That Poll

### FRAME 1: SETUP
**Visual:** Excited face
**Text:** "Let's settle this debate 👇"
**Music:** Dramatic sound

### FRAME 2: POLL
**Visual:** Split image comparing options
**Poll Sticker:**
"Morning person 🌅" vs "Night owl 🦉"

### FRAME 3: YOUR TAKE
**Visual:** Your face giving opinion
**Text:** "I'm a night owl BUT..."
**Text (small):** "Trying to switch because [reason]"

### FRAME 4: FOLLOW-UP
**Visual:** Results teaser
**Text:** "I'll share results + why it matters tomorrow"
**Question Sticker:** "Which are you and why?"
```
