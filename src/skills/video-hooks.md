# Video Hooks Skill

Master the science of attention. Create hooks that stop the scroll, trigger curiosity, and keep viewers watching. Based on 2026 research from platform analytics and creator success patterns.

> **"You have 3 seconds. Make them count."**

## When to Invoke

Invoke this skill when:
- Writing opening lines for video content
- Creating social media post hooks
- Developing ad creative
- Optimizing content for retention
- A/B testing opening strategies

---

## 🧠 The Psychology of Hooks

### Why Hooks Matter

**The 3-Second Reality:**
- Mobile users decide in 0.5-3 seconds
- Long-form viewers decide in 8 seconds
- 20% of viewers drop off in first 30 seconds
- Strong hooks improve retention by 2-5x

### What Triggers a Hook

**Dopamine Triggers:**
1. **Curiosity** - Open loops, unanswered questions
2. **Surprise** - Pattern interrupts, unexpected claims
3. **Identity** - "If you're a [specific person]..."
4. **Emotion** - Fear, excitement, anger, joy
5. **Promise** - Clear benefit in exchange for time

## 🎣 HOOK FORMULA LIBRARY

### Category 1: Curiosity Hooks

**The Curiosity Gap**
```
"What if I told you [surprising claim]?"
"The one thing [authority] doesn't want you to know..."
"Why 90% of [people] fail at [thing]..."
"I bet you didn't know this about [topic]..."
```

**The Secret Reveal**
```
"Nobody talks about this, but..."
"Here's something [experts] won't tell you..."
"The hidden truth about [topic]..."
"What they don't teach you about [thing]..."
```

**The Unexpected Connection**
```
"What [unrelated thing A] taught me about [thing B]..."
"The [unlikely source] hack that changed my [area]..."
"Why [random thing] is the key to [goal]..."
```

### Category 2: Story Hooks

**In Media Res (Start at the Climax)**
```
"I was sitting in my car, crying."
"Three years ago, I almost gave up."
"The email said: 'You're fired.'"
"$50,000 gone. In one day."
```

**The Transformation Setup**
```
"[Time] ago, I was [bad situation]."
"I went from [before] to [after] in [time]."
"This is what [time] of [action] did for me."
"When I started, I couldn't even [basic thing]."
```

**The Mistake Story**
```
"I wasted [X years/dollars] making this mistake."
"The $[amount] lesson I'll never forget."
"Don't do what I did..."
"If I could go back, I'd change this one thing."
```

### Category 3: Contrarian Hooks

**The Direct Challenge**
```
"[Common advice] is wrong. Here's why."
"Stop doing [popular thing]."
"Everything you know about [topic] is a lie."
"[Authority] is lying to you about [topic]."
```

**The Unpopular Opinion**
```
"Unpopular opinion: [controversial take]"
"I know I'll get hate for this, but..."
"Hot take: [statement]"
"Controversial: [claim]"
```

**The Myth Buster**
```
"[Common belief] is a myth."
"This is why [popular advice] doesn't work."
"[X] isn't what you think it is."
```

### Category 4: Result Hooks

**The Specific Achievement**
```
"I made $[X] in [timeframe] doing [thing]."
"How I went from [before] to [after] in [time]."
"This got me [specific result]."
"[X] changed everything for me."
```

**The Proof-First**
```
"Here's my [result]. Let me show you how."
"[Screenshot/proof]. This is real."
"I'll prove it to you right now."
```

**The Framework Reveal**
```
"The exact [system/method] I use to [achieve result]."
"My [X]-step process that [result]."
"The framework that [famous person] uses."
```

### Category 5: Direct Value Hooks

**The Promise**
```
"In the next [X] seconds/minutes, you'll learn [benefit]."
"By the end of this, you'll know [skill/info]."
"Watch this if you want to [achieve goal]."
"This will save you [time/money]."
```

**The Listicle Tease**
```
"[X] things that [changed/ruined] my [area]."
"[X] reasons why [surprising claim]."
"The [X] biggest mistakes [people make]."
"[X] [things] I wish I knew sooner."
```

**The How-To**
```
"Here's how to [achieve result] in [time]."
"The fastest way to [goal]."
"How to [do thing] without [pain point]."
```

### Category 6: Pattern Interrupt Hooks

**Visual Interrupts**
```
[Start mid-action]
[Unexpected visual]
[Screen flash or movement]
[Object dropping into frame]
```

**Audio Interrupts**
```
[Start with sound effect]
[Unexpected silence then speech]
[Music cut to voice]
[Loud → quiet shift]
```

**Format Interrupts**
```
"Wait— before we start..."
"Okay, this is going to sound crazy, but..."
"Stop. I need to tell you something."
"This wasn't supposed to happen, but..."
```

### Category 7: Identity Hooks

**The Call-Out**
```
"If you're a [specific person], this is for you."
"[Specific group], stop scrolling."
"To all the [identity group] out there..."
"Fellow [identity]— listen up."
```

**The Relatable Situation**
```
"POV: You're [relatable scenario]"
"When [relatable situation] happens..."
"You know that feeling when [experience]?"
"Ever had [common experience]? Here's why."
```

### Category 8: Challenge Hooks

**The Dare**
```
"I bet you can't [do thing]."
"Try this for [X] days."
"Most people can't do this for [time]."
"Challenge: [action]"
```

**The Test**
```
"Let's see if you know this..."
"Quick— can you answer this?"
"Test your [knowledge/skill]:"
```

## 📱 PLATFORM-SPECIFIC HOOKS

### YouTube (Long-Form)

**Best Hook Length:** 5-15 seconds

**Structure:**
```
[0-3 sec] Pattern interrupt or bold claim
[3-8 sec] Promise or curiosity gap
[8-15 sec] Credibility + preview
```

**Example:**
```
"What if I told you everything you know about productivity is wrong?
In the next 10 minutes, I'm going to show you the system that
top CEOs actually use—and it's nothing like what the gurus teach.
I spent 6 months researching this. Let's dive in."
```

### TikTok/Reels (Short-Form)

**Best Hook Length:** 1-3 seconds

**Structure:**
```
[0-1 sec] Visual/audio hook
[1-3 sec] Text + verbal hook
```

**Example:**
```
[VISUAL: Quick zoom in on face]
[TEXT: "Stop doing this"]
"If you're still [common mistake], you're wasting your time."
```

### YouTube Shorts

**Best Hook Length:** 2-5 seconds

**Structure:**
```
[0-2 sec] Hook statement
[2-5 sec] Quick context + promise
```

**Example:**
```
"3 apps that'll make you $500/month. Number 2 made me $2,000 last month alone."
```

### LinkedIn

**Best Hook Length:** First line (visible before "see more")

**Structure:**
```
[Line 1] 6-8 word hook that creates curiosity
[Line 2-3] Context that makes them click "see more"
```

**Example:**
```
I got rejected from 47 jobs in 3 months.

47.

↓
```

### Twitter/X

**Best Hook Length:** First sentence of tweet

**Structure:**
```
[Opening] Bold claim or result
[Follow-up] "Here's how:" or context
```

**Example:**
```
I made $50K from one Twitter thread.

Here's the exact framework (steal this):
```

## 🔬 HOOK TESTING FRAMEWORK

### A/B Test These Elements

1. **Opening word/phrase**
   - Question vs. Statement
   - "I" vs. "You"
   - Number vs. Word

2. **Specificity level**
   - Vague: "A lot of money"
   - Specific: "$47,382"

3. **Emotion**
   - Fear vs. Excitement
   - Curiosity vs. Promise

4. **Format**
   - Text-first vs. Visual-first
   - Story vs. Direct

### Hook Scorecard

Rate each hook 1-5 on:

| Criteria | Score |
|----------|-------|
| **Curiosity** - Does it create an open loop? | /5 |
| **Specificity** - Is it concrete? | /5 |
| **Relevance** - Does audience care? | /5 |
| **Believability** - Is it credible? | /5 |
| **Urgency** - Why watch NOW? | /5 |
| **Total** | /25 |

**Target:** 18+ before publishing

## 🚫 HOOKS TO AVOID

### What Doesn't Work

**Too Vague:**
```
❌ "This is really good..."
❌ "You need to see this..."
❌ "Important information..."
```

**No Curiosity:**
```
❌ "In this video, I'll talk about [topic]..."
❌ "Today we're going to discuss..."
❌ "Welcome to my channel..."
```

**Clickbait Without Payoff:**
```
❌ "You won't BELIEVE..." (without delivering)
❌ Sensational claims you can't back up
❌ Bait-and-switch content
```

**Too Long Before Value:**
```
❌ 30+ second intros
❌ Logos/animations before hook
❌ "Before we start, make sure to subscribe..."
```

## ✅ HOOK CHECKLIST

Before finalizing any hook:

- [ ] Can you say it in under 3 seconds?
- [ ] Does it create curiosity or promise value?
- [ ] Is it specific (not vague)?
- [ ] Would YOU stop scrolling?
- [ ] Does it match the content that follows?
- [ ] Is there a visual component to support it?
- [ ] Have you created 3 variations to test?

---

## 🔄 HOOK ITERATION PROCESS

1. **Write 5 hook variations**
2. **Score each using the scorecard**
3. **Select top 3 for testing**
4. **Film/post all three**
5. **Measure retention at 3 seconds**
6. **Double down on winner**
