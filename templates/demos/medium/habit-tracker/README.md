# Habit Tracker Demo

A daily habit tracking app with streaks, charts, and reminders.

## Overview

Build a personal productivity app that helps users build and maintain positive habits through daily tracking, streak counting, and visual progress charts.

## Target Audience

- People wanting to build better habits
- Self-improvement enthusiasts
- Productivity-focused individuals
- Health and fitness tracking

## Complexity

**Medium** - 1 day

## Key Features

1. **Habit Management**
   - Create custom habits
   - Set frequency (daily, weekly, specific days)
   - Habit categories (health, productivity, learning)
   - Habit icons and colors
   - Archive/delete habits

2. **Daily Tracking**
   - Check off completed habits
   - Today view with all habits
   - Quick add notes to completions
   - Partial completion (percentage)
   - Skip/excuse option

3. **Streaks & Motivation**
   - Current streak counter
   - Longest streak record
   - Streak freeze (1 miss allowed)
   - Milestone celebrations
   - Daily motivational quotes

4. **Progress Analytics**
   - Weekly completion calendar (GitHub-style)
   - Monthly view
   - Habit-specific charts
   - Overall completion rate
   - Best performing habits

5. **Reminders**
   - Push notifications (web/mobile)
   - Custom reminder times per habit
   - Smart reminders (based on completion patterns)

## MVP Scope (for 1 day build)

Focus on:
- Habit CRUD
- Daily check-in interface
- Streak calculation
- 7-day completion view
- Basic statistics

## Success Metrics

- User can track 5+ habits daily
- Streak calculation 100% accurate
- App usable in < 30 seconds daily

## Example Use Cases

1. Morning routine tracking (meditation, exercise, journaling)
2. Learning habits (reading, practicing skills)
3. Health habits (water intake, sleep, steps)
4. Productivity habits (inbox zero, deep work blocks)
