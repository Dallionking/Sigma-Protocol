# Event RSVP Demo

An event management platform with ticketing, RSVPs, and check-in.

## Overview

Build an event management platform for creating event pages, selling tickets, managing RSVPs, and checking in attendees at the door.

## Target Audience

- Event organizers
- Conference hosts
- Meetup organizers
- Workshop facilitators
- Community managers

## Complexity

**Medium** - 1-2 days

## Key Features

1. **Event Creation**
   - Event details (title, date, location)
   - Rich description with images
   - Multiple ticket types
   - Capacity limits
   - Event cover image

2. **Ticketing**
   - Free and paid tickets
   - Early bird pricing
   - Promo codes
   - Quantity limits
   - Waitlist when sold out

3. **Registration/RSVP**
   - Simple registration form
   - Custom questions
   - Confirmation emails
   - Calendar invite (.ics)
   - QR code ticket

4. **Attendee Management**
   - Attendee list with filters
   - Export attendee data
   - Send announcements
   - Refund processing

5. **Check-in**
   - Mobile check-in app
   - QR code scanning
   - Manual search check-in
   - Real-time attendance count

## MVP Scope (for 1-2 day build)

Focus on:
- Event creation page
- Public event page
- Free registration flow
- Email confirmation
- Basic attendee list
- Simple check-in

## Success Metrics

- Event created in < 10 minutes
- Registration < 2 minutes
- Check-in < 5 seconds per person

## Example Use Cases

1. Tech meetup with free tickets
2. Workshop with limited paid seats
3. Conference with multiple ticket tiers
4. Private corporate event
