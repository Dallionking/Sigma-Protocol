# User Stories: WriteStack

## Epic 1: Publication Setup

### US-1.1: Create Publication
**As a** writer
**I want to** create my publication
**So that** I can start publishing

### US-1.2: Customize Branding
**As a** writer
**I want to** add logo and colors
**So that** my publication is branded

### US-1.3: Write About Page
**As a** writer
**I want to** create an about page
**So that** readers know who I am

### US-1.4: Connect Stripe
**As a** writer
**I want to** connect payment processing
**So that** I can accept paid subscriptions

### US-1.5: Set Pricing
**As a** writer
**I want to** set subscription prices
**So that** readers can pay

---

## Epic 2: Content Creation

### US-2.1: Create New Post
**As a** writer
**I want to** write a new post
**So that** I can publish content

### US-2.2: Use Rich Text Editor
**As a** writer
**I want to** format my content
**So that** it looks professional

### US-2.3: Add Images
**As a** writer
**I want to** include images
**So that** my posts are visual

### US-2.4: Embed Media
**As a** writer
**I want to** embed videos/tweets
**So that** my posts are engaging

### US-2.5: Save Draft
**As a** writer
**I want to** save without publishing
**So that** I can continue later

### US-2.6: Preview Post
**As a** writer
**I want to** preview before publishing
**So that** I catch errors

**As a** writer

### US-3.5: Unpublish Post
**As a** writer

**I want to** segment subscribers

### US-5.4: Export Subscribers
