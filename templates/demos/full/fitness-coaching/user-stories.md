# User Stories: FitCoach

## Epic 1: Trainer Onboarding

### US-1.1: Create Trainer Account
**As a** personal trainer
**I want to** create an account
**So that** I can start coaching clients

### US-1.2: Setup Trainer Profile
**As a** trainer
**I want to** complete my profile
**So that** clients see my credentials

### US-1.3: Get Shareable Profile URL
**As a** trainer
**I want to** a custom URL
**So that** I can share my profile

---

## Epic 2: Exercise Library

### US-2.1: Browse Exercise Library

### US-2.2: Search Exercises

### US-2.3: Add Custom Exercise
