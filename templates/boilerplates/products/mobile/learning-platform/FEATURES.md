# Learning Platform -- Feature Breakdown

## 1. Authentication System

- Email/password sign-up with validation
- Email verification (OTP input)
- Sign-in with credentials
- OAuth callback handling (Google, Apple)
- Two-factor authentication (2FA)
- Forgot password flow (email, check email, reset)
- Success confirmation screens

## 2. Onboarding Flow

- Welcome screen with branding
- Goal selection (career, personal, academic, hobby)
- Level assessment (beginner through advanced)
- Motivation selection (why learn)
- Fast win exercise (immediate value)
- Fast win success celebration
- Notification opt-in
- Completion screen

## 3. Home / Community Feed

- Pinned announcements
- Lesson posts from AI Tutor
- Community text posts
- Post detail with comments
- Create post
- Like/comment interactions
- Homework assignments with submissions
- Daily challenge
- Weekly challenge
- Verb/concept of the day
- Streak detail and calendar

## 4. Lesson Library

- Category list (grid layout)
- Lesson list per category
- Lesson content viewer with sections
- Lesson audio player
- Vocabulary list and flashcard deck
- Vocabulary detail with pronunciation
- Story list and reader
- Worksheet viewer (PDF)
- Slang/jargon regions and phrases
- Lesson completion with XP reward
- Lesson locked state (tier gating)
- Assessment quiz

## 5. Practice Exercises

- Exercise type selection
- Multiple choice quiz (MCQ)
- Fill in the blank
- Speaking practice (pronunciation)
- Sentence builder (word tiles)
- Listening comprehension
- Translation (bidirectional)
- Timed drill (speed round)
- Pronunciation scoring
- Result summary with stats
- Result detail per question

## 6. AI Tutor

- Mode selection (5 modes)
- Free chat conversation
- Grammar focus with corrections
- Story mode (collaborative)
- Quick drill (rapid-fire Q&A)
- Voice chat with waveform
- Voice listening state
- Voice response playback
- Session summary with XP
- Typing indicator
- Correction highlighting
- Chat bubble system

## 7. Tutor Scheduling

- Calendar grid view
- Time slot picker
- Booking confirmation
- Booking success
- Upcoming sessions list
- Past sessions history
- Session detail
- Video room (LiveKit stub)
- Video ended summary

## 8. Subscription / Monetization

- Paywall with tier cards
- Plan comparison table
- Feature matrix (content, AI, speaking, community, support)
- Bonus value stack
- Billing toggle (monthly/annual)
- Checkout flow
- Success confirmation
- Subscription management
- Cancellation flow with retention offer
- 90-day guarantee badge

## 9. Profile & Settings

- Profile header with avatar
- Edit profile
- Stats dashboard (XP, lessons, words, hours)
- Weekly activity chart
- Streak calendar
- Achievements list (unlocked, in-progress, locked)
- Settings menu
- Account settings (email, password)
- Notification preferences
- Language & audio settings
- Appearance settings
- Privacy settings
- Help & support with FAQ
- Contact form

## 10. Gamification System

- XP points per lesson/practice/streak
- Level progression
- Streak tracking with calendar
- Streak badges
- Achievement categories (lessons, streak, vocabulary, speaking, special)
- XP progress ring
- Animated counter
- Confetti overlay (celebrations)

## 11. State Screens

- Error: generic, server, offline, access denied, session expired
- Empty: no lessons, no achievements, no bookings

## 12. Launch Screens

- Splash screen
- Maintenance mode
- Update required
